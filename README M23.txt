# Análisis del Titanic

Este proyecto consiste en la exploración y visualización de datos del famoso dataset del Titanic, con el objetivo de analizar las características de los pasajeros y su relación con la supervivencia.

## Descripción del Dataset

El dataset `titanic.csv` contiene información de 887 pasajeros e incluye las siguientes columnas:

- `Survived`: Indica si el pasajero sobrevivió (1) o no (0).
- `Pclass`: Clase del pasajero (1, 2 o 3).
- `Name`: Nombre del pasajero.
- `Sex`: Género del pasajero (male/female).
- `Age`: Edad del pasajero.
- `Siblings/Spouses Aboard`: Número de hermanos/esposos a bordo.
- `Parents/Children Aboard`: Número de padres/hijos a bordo.
- `Fare`: Tarifa pagada por el pasajero.

## Objetivos

1. Explorar los datos para entender la distribución de los pasajeros según clase, edad y género.
2. Analizar qué variables pueden estar relacionadas con la supervivencia.
3. Visualizar patrones mediante gráficos para obtener conclusiones preliminares.

## Requisitos

- Python 3.x
- Pandas
- Matplotlib
