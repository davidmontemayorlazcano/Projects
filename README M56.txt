El objetivo de este proyecto es construir un modelo de series de tiempo para pronosticar las temperaturas promedio diarias en Nueva York, utilizando el conjunto de datos TempNY.csv. El análisis se centra en la aplicación de modelos de pronóstico y la evaluación de su precisión.

Contenido del Repositorio
Tarea M56-CD – David Montemayor.ipynb: El notebook de Jupyter que contiene todo el código, el análisis y los gráficos de la tarea.

TempNY.csv: El conjunto de datos de temperaturas utilizado para el pronóstico.

README.md: Este archivo, que describe el proyecto.

Requisitos para la Ejecución
Para ejecutar el notebook, necesitarás tener instaladas las siguientes bibliotecas de Python:

pandas

numpy

matplotlib

statsmodels
