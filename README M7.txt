Iris Dataset Analysis
Descripción

Este proyecto realiza un análisis exploratorio y filtrado del famoso Iris dataset, que contiene mediciones de sépalos y pétalos de tres especies de iris: Iris-setosa, Iris-versicolor e Iris-virginica.

El análisis incluye:

Visualización de datos y selección de columnas.

Filtrado de datos según condiciones específicas.

Estadísticas descriptivas y estandarización de variables.
