Tarea de Regresión Lineal para Predicción de Precios de Viviendas
Este repositorio contiene la solución a la Tarea M52-CD del curso de Ciencia de Datos. El objetivo de este proyecto es construir un modelo de regresión lineal para predecir los precios de viviendas utilizando el conjunto de datos kc_house_data.csv. El proyecto compara los resultados de un modelo creado de forma manual con uno generado automáticamente para evaluar la efectividad de ambos enfoques.

Contenido del Repositorio
Tarea M52-CD – David Montemayor.ipynb: El notebook de Jupyter que contiene todo el código y el análisis de la tarea.

kc_house_data.csv: El conjunto de datos utilizado para entrenar y evaluar el modelo.
