# Consulta SQL: Empleados Técnicos por Fecha de Nacimiento

## Objetivo
Obtener los registros de empleados con el puesto de "Technician" cuyo año de nacimiento esté entre 1980 y 1995, ordenados de manera descendente por fecha de nacimiento.

## Descripción
Se utiliza una consulta SQL que combina los comandos:
- `SELECT *` para obtener todas las columnas de la tabla.
- `WHERE` con `LIKE` para filtrar por el puesto de trabajo.
- `BETWEEN` para seleccionar empleados dentro del rango de fechas.
- `ORDER BY` para ordenar los resultados por fecha de nacimiento de forma descendente.

## Tecnología / Librerías
- SQL Server / T-SQL

## Uso
Ejecutar la consulta en el entorno de base de datos correspondiente.
