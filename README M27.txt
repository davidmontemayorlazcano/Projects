Este repositorio contiene la solución a la tarea de Clustering K-Means para el curso de Ciencia de Datos. El objetivo de este proyecto es realizar el agrupamiento del conjunto de datos Iris.

Contenido del Repositorio
Tarea M27-CD – David Montemayor.ipynb: El notebook de Jupyter que contiene el código para el análisis y la visualización del método del codo y el coeficiente de silueta.

Iris3.csv: El conjunto de datos de entrada utilizado en el análisis.

Requisitos para la Ejecución
Para ejecutar el notebook, necesitas tener instaladas las siguientes bibliotecas de Python:

numpy

pandas

matplotlib

seaborn

scikit-learn
