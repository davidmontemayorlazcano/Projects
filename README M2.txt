Conteo de Palabras en una Oración
Descripción

Este proyecto consiste en un programa en Python que recibe una oración y genera un diccionario con la frecuencia de cada palabra. Posteriormente, imprime cuántas veces aparece cada término en el texto.

Tecnologías usadas

Python 3.9

Jupyter Notebook

Cómo ejecutar el código

Abrir el archivo .ipynb en Jupyter Notebook.

Ejecutar las celdas del notebook.

Observar en pantalla el conteo de palabras y sus repeticiones.
