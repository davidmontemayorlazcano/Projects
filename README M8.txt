NumPy Basic Operations
=====================

Descripcion
-----------
Este proyecto demuestra el uso basico de NumPy, la biblioteca fundamental de Python para computacion cientifica, mediante ejercicios practicos que incluyen:

- Creacion y manipulacion de arrays.
- Inversion de arrays.
- Conversion de temperaturas de Fahrenheit a Celsius.
- Concatenacion de arrays y eliminacion de duplicados.

Requisitos
----------
- Python 3.x
- NumPy

Instalacion de NumPy:
pip install numpy

Uso
---
1. Clonar o descargar el repositorio.
2. Abrir el notebook en Jupyter Notebook.
3. Ejecutar las celdas paso a paso para observar los resultados.
