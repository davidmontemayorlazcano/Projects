El objetivo de este proyecto es construir un modelo de regresión lineal para pronosticar los precios de las acciones de Disney (DIS), basándose en datos históricos. El análisis se basa en datos obtenidos de yfinance y se evalúa la precisión del modelo utilizando métricas como el Error Cuadrático Medio (RMSE) y el Error Porcentual Absoluto Medio (MAPE).
