El objetivo de este proyecto es construir un modelo de regresión lineal para predecir las ventas (Sales) de una campaña publicitaria, utilizando variables como la inversión en televisión (TV), radio (Radio) y periódicos (Newspaper). El proyecto se enfoca en la estimación manual de los coeficientes del modelo y la validación de sus supuestos.

Contenido del Repositorio
Tarea M53-CD – David Montemayor.ipynb: El notebook de Jupyter que contiene todo el código y el análisis de la tarea.

Advertising.csv: El conjunto de datos utilizado para entrenar el modelo.

Requisitos para la Ejecución
Para ejecutar el notebook, necesitas tener instaladas las siguientes bibliotecas de Python:

pandas

numpy

statsmodels
