Descripción
Este proyecto aplica técnicas de análisis de datos y aprendizaje no supervisado para segmentar clientes de Amazon según sus características de compra. Se utiliza clustering jerárquico para identificar patrones y agrupar compradores con comportamientos similares.

Objetivo
Determinar grupos de clientes con preferencias similares para recomendar productos personalizados, mejorando así la experiencia de compra.

Tecnologías utilizadas
Python

Pandas

NumPy

Matplotlib

Scikit-learn

SciPy

Proceso
Carga y limpieza de datos desde un archivo Excel

Normalización de variables numéricas

Visualización con dendogramas

Determinación del número óptimo de clusters

Aplicación de Agglomerative Clustering

Reducción de dimensionalidad con PCA

Recomendaciones personalizadas basadas en similitud de grupo

Cómo ejecutar
Asegúrate de tener los paquetes necesarios instalados (pandas, numpy, matplotlib, scikit-learn, scipy)

Coloca el archivo Amazon.xlsx en el mismo directorio del script

Ejecuta el script en un entorno como Jupyter Notebook o VSCode
