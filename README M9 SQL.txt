El objetivo de este proyecto es demostrar un amplio conocimiento en el uso de diferentes tipos de uniones (Joins) y otras operaciones avanzadas para combinar, filtrar y manipular datos en una base de datos SQL.

Objetivo del Proyecto
El script se enfoca en la aplicación práctica de comandos de unión para relacionar información entre diferentes tablas. Además, se exploran otros operadores clave como UNION, UNION ALL y la instrucción CASE para la creación de reportes y la categorización de datos.

Comandos SQL y Operadores Utilizados
INNER JOIN: Une registros de dos tablas que tienen valores coincidentes.

LEFT OUTER JOIN: Retorna todos los registros de la tabla izquierda y los registros coincidentes de la tabla derecha.

RIGHT OUTER JOIN: Retorna todos los registros de la tabla derecha y los registros coincidentes de la tabla izquierda.

FULL OUTER JOIN: Retorna todos los registros cuando hay una coincidencia en una de las tablas.

CROSS JOIN: Retorna el producto cartesiano de las tablas involucradas, es decir, todas las combinaciones posibles de las filas.

UNION y UNION ALL: Combinan los conjuntos de resultados de dos o más consultas SELECT en un único conjunto de resultados. UNION elimina duplicados, mientras que UNION ALL no lo hace.

INSTRUCCIÓN CASE: Permite evaluar una lista de condiciones y retornar un valor en función de la primera condición que se cumpla.
