Conversión de Números Romanos a Enteros
Descripción

Este proyecto en Python implementa una clase llamada Conversion, la cual contiene un método para transformar un número romano en su equivalente en número entero.

El algoritmo recorre la cadena de derecha a izquierda, aplicando las reglas de conversión de números romanos:

Si el valor actual es mayor o igual al anterior, se suma.

Si es menor, se resta.

Tecnologías usadas

Python 3.9

Jupyter Notebook

Cómo ejecutar el código

Abrir el archivo .ipynb en Jupyter Notebook.

Definir la clase Conversion.

Llamar al método romano_entero() con un número romano como parámetro.
