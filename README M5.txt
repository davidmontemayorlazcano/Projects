Funciones: Invertir Cadenas y Verificar Números
Descripción

Este proyecto en Python contiene dos funciones principales:

invierte_cadena(texto)

Recibe una cadena de texto como parámetro.

Devuelve la cadena invertida utilizando slicing.

prueba_primo(numero)

Recibe un número como parámetro.

Devuelve True si el número es par y False en caso contrario (nota: aunque la función se llama "prueba_primo", realmente verifica si el número es par).

Tecnologías usadas

Python 3.9

Jupyter Notebook

Cómo ejecutar el código

Abrir el archivo .ipynb en Jupyter Notebook.

Ejecutar las celdas.

Probar la función invierte_cadena con cualquier texto y la función prueba_primo con cualquier número entero.
