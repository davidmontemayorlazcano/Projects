Tarea de Subconsultas, CTEs y Funciones de Cadena en SQL
Este repositorio contiene la solución a la Tarea M11 del curso de Ciencia de Datos. El objetivo de este proyecto es demostrar habilidades en la aplicación de técnicas avanzadas de consulta en SQL, incluyendo el uso de subconsultas, Expresiones de Tabla Comunes (CTEs), tablas temporales y funciones de cadena.

Objetivo del Proyecto
El script aborda diferentes escenarios de consulta:

Subconsultas (IN): Recuperar productos que han sido vendidos en al menos una orden.

CTEs (WITH): Simplificar consultas complejas al segmentarlas en partes lógicas para obtener el nombre de empleados y el departamento al que pertenecen.

Tablas Temporales (# y ##): Crear estructuras de datos temporales para almacenar y manipular datos de forma eficiente, como una lista de productos que han sido ordenados.

Funciones de Cadena (SUBSTRING): Extraer información específica de cadenas de texto, como números de orden de ventas.

Comandos y Conceptos SQL Utilizados
Subconsultas: Una consulta anidada dentro de otra consulta.

WITH: Define una Expresión de Tabla Común (CTE) que se puede referenciar dentro de una sentencia SELECT, INSERT, UPDATE o DELETE.

Tablas Temporales: Tablas que solo existen durante la sesión actual de la base de datos o para un período de tiempo definido.

SUBSTRING: Una función que extrae una subcadena de una cadena de caracteres.
