El objetivo de este proyecto es construir un modelo de regresión lineal para predecir los precios de las acciones de las empresas Walmart (WMT) e IBM (IBM). El análisis se basa en datos históricos de 5 años obtenidos de yfinance y se evalúa la precisión del modelo utilizando métricas como el Error Cuadrático Medio (RMSE) y el Error Porcentual Absoluto Medio (MAPE).
